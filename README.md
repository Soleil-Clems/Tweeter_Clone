# Tweeter_Clone


Ce projet est un clone de Twitter, développé à l'aide des technologies suivantes : Sass, HTML, jQuery, Ajax, PHP, MVC, et POO (Programmation Orientée Objet).

## Fonctionnalités

- **Inscription et Connexion :** Les utilisateurs peuvent créer un compte et se connecter à leur compte existant.
- **Publication de Tweets :** Les utilisateurs peuvent écrire et publier des tweets.
- **Abonnements :** Les utilisateurs peuvent suivre d'autres utilisateurs et voir leurs tweets dans leur fil d'actualité.
- **Profil Utilisateur :** Chaque utilisateur a un profil où ses tweets sont affichés et où il peut modifier ses informations personnelles.
- **Recherche :** Les utilisateurs peuvent rechercher d'autres utilisateurs et des tweets spécifiques.
- **Notifications :** Les utilisateurs reçoivent des notifications pour les nouveaux abonnés, les mentions et les messages directs.
- **Système de Like :** Les utilisateurs peuvent aimer les tweets d'autres utilisateurs.
- **Système de Commentaires :** Les utilisateurs peuvent commenter les tweets des autres utilisateurs.
- **Messagerie :** Les utilisateurs peuvent envoyer des messages directs à d'autres utilisateurs.

## Structure du Projet

Le projet suit le modèle MVC (Modèle-Vue-Contrôleur) pour une organisation claire du code :

- **Modèles (Models) :** Gèrent l'accès aux données et la logique métier.
- **Vues (Views) :** Génèrent l'interface utilisateur et affichent les données.
- **Contrôleurs (Controllers) :** Traitent les requêtes de l'utilisateur, récupèrent les données nécessaires et les transmettent aux vues.
